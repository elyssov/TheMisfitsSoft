# The Misfits Software

Independent static campaign site for The Misfits Software.

## Local preview

Serve the repository root with any static web server. No build step is required.

## Deployment

Every push to `main` deploys the repository through GitHub Pages. The canonical
entry point is `https://lyssovsky.com/themisfitssoftware/`, which redirects to
the Pages deployment while the project remains independently portable.
