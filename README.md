# The Misfits Software

**A human-led, AI-powered independent game studio building authored games and
open production tooling without waiting for permission.**

[Visit the studio site](https://lyssovsky.com/themisfitssoftware/)

This repository is the public, open-source studio shell for The Misfits
Software: a portable static campaign site, a transparent production
constitution, and a map of the studio's projects and public artifacts. It has
no build step, no framework dependency, and no backend requirement.

## The first promise

The studio has four designed worlds, but only one current production promise:
the **Mystery of Ancient Darkness (MOAD)** vertical slice.

MOAD is a 1930s occult adventure about Lord Eugene and Lady Alice Compton, two
parents entering ancient darkness to recover their nine-year-old son. The
public prologue player is available here:

- [Open-source repository](https://github.com/elyssov/MOAD-Demo)
- [Playable prologue](https://elyssov.github.io/MOAD-Demo/)

The wider slate also includes *Tanks of Thea*, *FAKEL / ORPP: Inferno*, and
*AEGIS: Cold Iron*. They are published as directions, not promises competing
for MOAD's production resources.

## What is open here

- The dependency-free HTML/CSS campaign site
- The studio's public production constitution
- The four-project slate and current scope commitment
- The AI-assistance disclosure and contributor model
- A reusable foundation for future project and release pages

The site is deliberately static and portable: clone it, serve the repository
root, and it works. This keeps the public surface auditable and inexpensive for
a small team.

## Open-source roadmap

- Extract project data into a documented machine-readable manifest
- Add accessible navigation, reduced-motion behavior, and contrast checks
- Add English/Russian localization without a heavyweight framework
- Publish reusable campaign and studio-site templates
- Add automated link, metadata, and asset validation
- Connect public builds and release notes from each open project
- Document the human-directed, AI-assisted production workflow

## Production model

The Misfits Software is a small, unfunded, distributed garage collective. Human
authorship and direction remain responsible for canon, design, judgment, and
every public promise. Named AI collaborators assist with implementation,
research, review, localization, production planning, and continuity across
large bodies of project material.

The goal is not to mass-produce "AI content." It is to use AI collaboration to
give a very small team enough leverage to build specific, authored games and to
publish the useful tools and methods that emerge from that work.

## Local preview

Serve the repository root with any static HTTP server and open `index.html`.
No package installation or compilation is required.
